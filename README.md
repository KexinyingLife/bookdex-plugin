# 书籍角色文本图鉴（bookdex-plugin）

> 一个面向 Yunzai 的原神文本检索插件：书籍、角色故事、圣遗物、武器故事统一检索与阅读。

![help](./resources/help-main.jpg)

---

## 功能特性

- 📚 书籍图鉴：支持 `txt/docx` 导入、本地索引、标题/正文检索
- 👤 角色故事：从米游社图鉴抓取，支持详情/全文模式
- 🗿 圣遗物文本：套装与部件故事检索
- ⚔️ 武器故事：武器剧情文本检索
- 🔎 统一搜索：跨书籍/角色/圣遗物/武器全文搜索
- 🖼️ 图片阅读 + 纯文本阅读双模式（命令尾部加“文本”）

---

## 安装方式

在 **Yunzai 主目录**执行：

```bash
cd /opt/Yunzai
cd plugins
# 替换为你的仓库地址
git clone <你的仓库地址> bookdex-plugin
```

安装依赖（若未自动安装）：

```bash
cd /opt/Yunzai/plugins/bookdex-plugin
pnpm i
```

---

## 首次使用（必须先全量拉取）

首次下载后，请先执行一次全量更新：

```text
#统一更新
```

该命令会一次性更新：
- 书籍
- 角色故事
- 圣遗物
- 武器故事

---

## 指令帮助

### 数据更新

- `#统一更新`：全量更新（首次必做）
- `#书籍更新`：仅更新书籍
- `#角色故事更新`：仅更新角色故事
- `#圣遗物更新`：仅更新圣遗物
- `#武器更新`：仅更新武器故事
- `#书籍导入`：导入 `data/inbox` 中的 `txt/docx`

### 列表/帮助

- `#书籍帮助`
- `#角色故事帮助`
- `#圣遗物帮助`
- `#武器帮助`

### 阅读

- `#书名`
- `#书名文本`
- `#角色名故事`
- `#角色名故事详情`
- `#角色名故事文本`
- `#圣遗物名圣遗物`
- `#圣遗物名圣遗物文本`
- `#武器名武器故事`
- `#武器名武器故事文本`

> 引用帮助消息发送“序号”可快速读取；加“文本”返回纯文本。

### 搜索

- `#搜索 关键词`（全库）
- `#书籍搜索 关键词`
- `#角色故事搜索 关键词`
- `#圣遗物搜索 关键词`
- `#武器搜索 关键词`
