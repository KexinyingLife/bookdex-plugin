# 书籍角色文本图鉴（bookdex-plugin）

![help](./resources/help-main.jpg)

## 安装方式

在 Yunzai 主目录执行：

```bash
git clone https://github.com/KexinyingLife/bookdex-plugin.git ./plugins/bookdex-plugin
```

依赖安装（推荐，进入插件目录执行，适配所有用户环境）：

```bash
cd plugins/bookdex-plugin
pnpm i
```

---

## 首次使用（必须先全量拉取）

首次下载后先执行：

```text
#统一更新
```

该命令会一次更新：书籍、角色故事、角色语音、剧情文本、地图文本、圣遗物、武器故事。

---

## 自动拉取机制（42天周期）

插件内置了“42天一个周期”的自动更新窗口：

- 每天会检查一次窗口
- 在每个 42 天周期的第 **1~5 天**（不含第 0 天）自动执行统一更新
- 你仍可随时手动执行 `#统一更新`

---

## 使用说明

- 默认返回文本；命令带 `图片` 返回图片
- 建议优先使用文本模式，尽量少用图片模式
- 分类帮助、搜索结果、长文本会优先使用合并消息发送
- 受平台消息长度与合并消息上限影响，超长内容可能拆分发送，极端情况下会改为 `txt` 文件发送
- 受协议端限制与文本体量影响，图片模式在长文本场景下可能耗时较长，也可能出现发送失败或报错
- 只有引用插件自己发出的帮助、搜索、语音列表或结果消息，序号查看才会生效
- 由于米游社当前没有提供邀约任务文本入口，因此插件目前不支持邀约任务文本

示例：

```text

引用搜索结果发 3图片
引用语音列表发 2语音
```

## 数据来源（米游社观测枢）

本插件数据抓取使用米游社观测枢相关接口：

- 列表入口（selector）  
  `https://act-api-takumi.mihoyo.com/common/blackboard/ys_obc/v1/content/selector`
- 词条详情（entry_page）  
  `https://act-api-takumi-static.mihoyo.com/hoyowiki/genshin/wapi/entry_page`

频道参数：

- `channel_id=68`：书籍
- `channel_id=25`：角色故事 / 角色语音（语音数据来自角色词条内 `role_voice` 模块）
- `channel_id=43`：剧情文本（主要提取词条中的“剧情对话”模块）
- `channel_id=251`：地图文本（主要提取词条中的“交互文本/interactive_dialogue”模块）
- `channel_id=218`：圣遗物
- `channel_id=5`：武器
